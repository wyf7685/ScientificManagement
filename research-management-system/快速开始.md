# 快速开始指南

## 环境要求

- Node.js 16+ 
- npm 或 yarn

## 安装步骤

### 1. 安装依赖

```bash
cd research-management-system
npm install
```

### 2. 配置环境变量

如果暂时没有后端，可以开启前端内置 Mock 直接演示：

```env
VITE_USE_MOCK=false
```

若需要使用 Mock 数据进行开发，可以启用：

```env
VITE_API_BASE_URL=http://localhost:3000/api
VITE_USE_MOCK=true
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

## 功能导航

### 普通用户功能

1. **个人概览** (`/dashboard`)
   - 查看个人成果统计
   - 查看成果分布图表
   - 查看最近提交的成果

2. **创建成果物** (`/results/create`)
   - 选择成果类型
   - 填写基础信息
   - 智能补全（可选）
   - 上传附件
   - 设置可见范围

3. **个人成果物** (`/results/my`)
   - 查看个人所有成果
   - 筛选和搜索
   - 编辑和删除

4. **成果检索** (`/results/search`)
   - 高级检索
   - 查看成果详情

### 专家功能

除了普通用户的所有功能外，还包括：

5. **成果审核** (`/expert/reviews`)
   - 查看待审核成果
   - 审核操作（通过/驳回）
   - 填写审核意见

### 管理员功能

6. **科研看台** (`/admin/dashboard`)
   - 全局统计数据
   - 可视化图表
   - 最新入库成果

7. **成果管理** (`/admin/results`)
   - 全局成果列表
   - 筛选和管理

8. **审核分配** (`/admin/review-assign`)
   - 分配审核专家
   - 查看专家任务量

9. **成果类型配置** (`/admin/result-types`)
   - 管理成果类型
   - 配置动态字段

## 开发调试

### 查看网络请求

打开浏览器开发者工具（F12），切换到 Network 标签，可以查看所有 API 请求。

### 清除登录状态

在浏览器控制台执行 `localStorage.clear()` 即可。

## 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录，可以部署到任何静态文件服务器。

## 常见问题

### Q: 启动后页面空白？
A: 检查浏览器控制台是否有错误，确认 API 地址配置正确。

### Q: 登录后跳转到 403 页面？
A: 检查用户角色是否正确，路由权限配置是否匹配。

### Q: 图表不显示？
A: 确认 ECharts 已正确安装，检查数据格式是否正确。

### Q: 上传文件失败？
A: 检查后端上传接口是否正常，文件大小是否超限。
